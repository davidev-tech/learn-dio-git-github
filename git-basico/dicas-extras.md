## Comandos gerais do terminal Git Bash

| Comando        | Função                             |
| -------------- | ---------------------------------- |
| `ls`           | Listar arquivos/pastas             |
| `cat arquivo`  | Exibir conteúdo de um arquivo      |
| `pwd`          | Mostrar caminho do diretório atual |
| `echo "texto"` | Imprimir texto no terminal         |
