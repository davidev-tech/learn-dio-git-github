# 🖥️ Comandos do Terminal (Windows)

| Comando                    | Descrição                                               |
| -------------------------- | ------------------------------------------------------- |
| `dir`                      | Lista diretórios, arquivos e pastas do local atual      |
| `cd <nome>`                | Acessa uma pasta específica                             |
| `cd /`                     | Vai para o diretório raiz                               |
| `cd ..`                    | Volta um nível                                          |
| `cls`                      | Limpa a tela do terminal                                |
| `mkdir <nome>`             | Cria uma nova pasta (diretório)                         |
| `echo texto > arquivo.txt` | Cria um arquivo com o conteúdo; se existir, sobrescreve |
| `del <arquivo>`            | Deleta arquivos (não apaga a pasta)                     |
| `rmdir <pasta> /s /q`      | Remove a pasta e todo o seu conteúdo                    |
| `Tab`                      | Autocompletar nomes de arquivos/pastas                  |

> **Obs.:** Diretório e pasta são a mesma coisa. “Diretório” é o termo técnico mais usado em TI.
