# Histórico de Commits

```bash
git log                     # mostra o histórico completo. Pressione 'q' + Enter para sair
git log <diretorio>         # histórico apenas de um diretório
git log --oneline           # resume cada commit em uma linha
git log --graph             # exibe o histórico de forma visual e gráfica (grafo de commits)
gitk                        # interface gráfica ainda mais visual (abre uma janela separada)

Use `git log` com frequência para entender o que está acontecendo no repositório.
